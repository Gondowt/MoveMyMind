Solution










The computer chooses a pattern of four code pegs. Duplicates are allowed, so you could even choose four code pegs of the same color. A red peg is placed for each code peg from the guess which is correct in both color and position. A white key peg indicates the existence of a correct color code peg placed in the wrong position. Try to guess the computer pattern, in both order and color, within 15 turns.
